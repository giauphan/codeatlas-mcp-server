# Security Policy

## Reporting a Vulnerability

We take the security of CodeAtlas MCP Enterprise seriously. If you believe you have found a security vulnerability, please report it to **giauphan012@gmail.com** or open a [private security advisory](https://github.com/giauphan/codeatlas-mcp-server/security/advisories/new).

You should receive a response within 48 hours.

Please include:
- Type of issue
- Affected source files
- Steps to reproduce
- Impact of the issue

## Disclosure

Please do not publicly disclose the vulnerability until we have had the opportunity to address it.
